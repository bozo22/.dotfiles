## Setup Guide

### Basic configurations
```
git clone https://github.com/bozo22/.dotfiles ~/.dotfiles
cd .dotfiles
ln -s .bashrc ~/.bashrc
ln -s .vimrc ~/.vimrc
ln -s .Xresorces ~/.Xresources
ln -s neofetch/config.conf ~/.config/neofetch/config.conf
```
